# Getting Started

`@aknishi/akds-reactkit` is a React component library built on a design token system. Components adapt to light and dark themes out of the box.

---

## Installation

```sh
npm install @aknishi/akds-reactkit
```

---

## Setup

Wrap your application with `ThemeProvider` once at the root. This is the only setup step required — component styles are injected automatically when components are imported, so you don't need a separate CSS import.

```tsx
import { ThemeProvider } from '@aknishi/akds-reactkit';

export default function App() {
  return (
    <ThemeProvider defaultTheme="light">
      {/* your app */}
    </ThemeProvider>
  );
}
```

---

## Theming

`ThemeProvider` sets a `data-theme` attribute on `document.documentElement`, switching between the light and dark token sets. Use `useTheme` anywhere inside the tree to read or change the active theme.

```tsx
import { useTheme } from '@aknishi/akds-reactkit';

function ThemeToggle() {
  const { theme, setTheme } = useTheme();
  return (
    <button onClick={() => setTheme(theme === 'light' ? 'dark' : 'light')}>
      Switch to {theme === 'light' ? 'dark' : 'light'} mode
    </button>
  );
}
```

### Controlled mode

Pass `theme` and `onThemeChange` to let a parent component own the state:

```tsx
const [theme, setTheme] = React.useState('light');

<ThemeProvider theme={theme} onThemeChange={setTheme}>
  <App />
</ThemeProvider>
```

---

## Using components

Import components individually from the package root:

```tsx
import { Button, TextInput, Card } from '@aknishi/akds-reactkit';
```

All components are tree-shakeable — only the components you import will be included in your bundle. Styles are injected automatically alongside each component, so unused components carry no CSS overhead either.

---

## Icons

Icons are delivered as a separate package, `@aknishi/akds-icons`, so your app only pays for the icons it actually uses.

```sh
npm install @aknishi/akds-icons
```

Each icon is its own inline SVG React component sourced from [Material Symbols](https://fonts.google.com/icons) (Rounded style). No font file or CDN dependency is required.

```tsx
import { DeleteIcon, ChevronDownIcon, CheckCircleFilledIcon } from '@aknishi/akds-icons';

// Default size (md = 20px)
<DeleteIcon />

// With size
<ChevronDownIcon size="sm" />

// With semantic color
<CheckCircleFilledIcon size="lg" color="success" />

// As a standalone meaningful icon — provide aria-label
<DeleteIcon size="lg" aria-label="Delete item" aria-hidden={undefined} />
```
