# Spinner

> **Available from v1.0.0**

An animated loading indicator. Renders an SVG ring that rotates continuously. Used internally by `Button` during its loading state, and available standalone for any async feedback context.

## Usage

```tsx
import { Spinner } from '@aknishi/akds-reactkit';
```
