# RadioGroup

> **Available from v1.0.0**

A fieldset wrapper that manages a group of `Radio` inputs. Provides a `legend`, controls orientation (`vertical` / `horizontal`), and propagates a `disabled` state to all children. The `Radio` component handles individual option rendering with label, size, and disabled support.

## Usage

```tsx
import { RadioGroup, Radio } from '@aknishi/akds-reactkit';
```
