# Divider

> **Available from v1.0.2**

A visual separator between sections of content. Renders a semantic `<hr>` for plain dividers, or a `<div role="separator">` when a text label is provided. Supports horizontal and vertical orientations with solid, dashed, and dotted line styles.

## Usage

```tsx
import { Divider } from '@aknishi/akds-reactkit';
```
