# Checkbox

> **Available from v1.0.0**

A form control for binary or indeterminate boolean values. Supports `sm` and `md` sizes, a disabled state, and an indeterminate state for tri-state selection patterns (e.g. select-all).

## Usage

```tsx
import { Checkbox } from '@aknishi/akds-reactkit';
```
