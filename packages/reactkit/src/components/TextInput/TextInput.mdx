# TextInput

> **Available from v1.0.0**

A labelled text field with optional helper text, start adornment, and error state. The native `<input>` is forwarded a ref and accepts all standard HTML input attributes via `...rest`.

## Usage

```tsx
import { TextInput } from '@aknishi/akds-reactkit';
```
