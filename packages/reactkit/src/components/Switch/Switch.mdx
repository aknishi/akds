# Switch

> **Available from v1.0.2**

A toggle control for binary on/off settings. Renders an `<input type="checkbox" role="switch">` inside a styled track-and-thumb layout. Supports controlled and uncontrolled usage.

## Usage

```tsx
import { Switch } from '@aknishi/akds-reactkit';
```
