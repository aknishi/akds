# Flexbox

> **Available from v1.0.0**

A layout primitive that maps design tokens to CSS flexbox properties. Replaces inline styles for common layout patterns — direction, alignment, justification, gap, and spacing — while keeping all values tied to the token scale.

## Usage

```tsx
import { Flexbox } from '@aknishi/akds-reactkit';
```
