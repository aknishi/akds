# Text

> **Available from v1.0.0**

A typography primitive that maps a `styleAs` variant to the correct font size, weight, and line-height tokens. Decouples visual style from the rendered HTML element — use `as` to override the tag while keeping the intended style.

## Usage

```tsx
import { Text } from '@aknishi/akds-reactkit';
```
