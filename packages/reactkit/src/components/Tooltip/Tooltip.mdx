# Tooltip

> **Available from v1.0.0**

A non-interactive floating label that appears on hover or focus to provide supplementary context. Positioned relative to its trigger child and supports four placements: `top`, `right`, `bottom`, `left`. Renders in a portal to avoid clipping.

## Usage

```tsx
import { Tooltip } from '@aknishi/akds-reactkit';
```
