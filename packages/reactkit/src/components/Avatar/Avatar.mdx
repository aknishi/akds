# Avatar

> **Available from v1.0.2**

Displays a user's identity as an image, initials, or icon. Falls back through each variant in order: image → initials → provided icon → default person icon. Initials are derived from the first and last name (up to 3 characters) and a color is deterministically assigned from the name so the same person always gets the same color.

## Usage

```tsx
import { Avatar } from '@aknishi/akds-reactkit';
```
