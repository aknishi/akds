# Menu

> **Available from v1.0.0**

A floating action menu rendered in a portal. Controlled via `open` / `onOpenChange`, and optionally anchored to a trigger element via `triggerRef`. Supports keyboard navigation (Arrow keys, Home, End, Escape) and four placement options. Compose with `MenuItem` children.

## Usage

```tsx
import { Menu, MenuItem } from '@aknishi/akds-reactkit';
```
