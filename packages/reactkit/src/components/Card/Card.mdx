# Card

> **Available from v1.0.0**

A surface container that groups related content. Compose with `CardHeader`, `CardContent`, and `CardFooter` subcomponents to build structured layouts. All three are optional — use only what the design requires. Supports an optional `borderless` variant.

## Usage

```tsx
import { Card, CardHeader, CardContent, CardFooter } from '@aknishi/akds-reactkit';
```
