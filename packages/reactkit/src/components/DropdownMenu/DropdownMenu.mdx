# DropdownMenu

> **Available from v1.0.0**

A select input that opens a floating listbox of options. Supports single and multiple selection, an optional helper text, placeholder, and a disabled state. Compose with `Option` children to define the available choices.

## Usage

```tsx
import { DropdownMenu, Option } from '@aknishi/akds-reactkit';
```
