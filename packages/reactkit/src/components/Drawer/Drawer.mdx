# Drawer

> **Available from v1.0.0**

A panel that slides in from any edge of the viewport. Rendered into a portal, managed with an `open` prop, and closeable via the Escape key or clicking the scrim. Supports `left`, `right`, `top`, and `bottom` sides.

## Usage

```tsx
import { Drawer } from '@aknishi/akds-reactkit';
```
