# create-akds-app

Scaffolds a new **TypeScript + React + Vite** project pre-wired with the akds design system, React Router v7, and a light/dark theme toggle — ready to run in one command.

---

## Installation

```sh
npm create @aknishi/akds-app <project-name>
```

Or invoke directly with `npx`:

```sh
npx @aknishi/create-akds-app <project-name>
```

Then start the dev server:

```sh
cd <project-name>
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to see the app.

---

## What's included

- **TypeScript + React 18 + Vite** — instant HMR and optimised production builds
- **`@aknishi/akds-reactkit`** — the full component library, auto-styled via CSS variables
- **`@aknishi/akds-icons`** — inline SVG icon set
- **`@aknishi/akds-tokens`** — design token CSS variables
- **React Router v7** — wired up in `src/router.tsx`; add pages without extra config
- **`ThemeProvider`** — light/dark toggle in the Navbar out of the box
- **Landing page + About page** — two starter pages to build from

---

## Template structure

```
<project-name>/
├── src/
│   ├── App.tsx                  # Root — ThemeProvider + BrowserRouter
│   ├── App.css
│   ├── router.tsx               # Route definitions
│   ├── main.tsx
│   ├── components/
│   │   └── Navbar/              # Sticky header with brand, nav links, theme toggle
│   └── pages/
│       ├── LandingPage/         # Hero + feature cards + CTA
│       └── AboutPage/           # Simple info page
├── index.html
├── vite.config.ts
├── tsconfig.json
└── package.json
```
