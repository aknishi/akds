# Elevation

Shadow tokens create a sense of depth and hierarchy. Apply them via `box-shadow` using the CSS variables below.

---

## Shadow scale

<table>
  <thead>
    <tr><th>Level</th><th>CSS variable</th><th>Value</th></tr>
  </thead>
  <tbody>
    <tr><td><code>none</code></td><td><code>--akds-elevation-none</code></td><td><code>none</code></td></tr>

## Visual reference

<div style={{ display: 'flex', flexWrap: 'wrap', gap: 32, padding: '32px 0' }}>
  {[
    { level: 'none', shadow: 'none' },
    { level: 'xs',   shadow: '0 1px 2px rgba(0,0,0,0.05)' },
    { level: 'sm',   shadow: '0 1px 3px rgba(0,0,0,0.10), 0 1px 2px rgba(0,0,0,0.06)' },
    { level: 'md',   shadow: '0 4px 6px rgba(0,0,0,0.07), 0 2px 4px rgba(0,0,0,0.06)' },
    { level: 'lg',   shadow: '0 10px 15px rgba(0,0,0,0.10), 0 4px 6px rgba(0,0,0,0.05)' },
    { level: 'xl',   shadow: '0 20px 25px rgba(0,0,0,0.10), 0 10px 10px rgba(0,0,0,0.04)' },
    { level: '2xl',  shadow: '0 25px 50px rgba(0,0,0,0.25)' },
  ].map(({ level, shadow }) => (
    <div key={level} style={{ textAlign: 'center' }}>
      <div style={{
        width: 96,
        height: 96,
        background: 'var(--akds-color-surface-default, #fff)',
        borderRadius: 10,
        boxShadow: shadow,
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        marginBottom: 12,
        border: level === 'none' ? '1px solid rgba(0,0,0,0.08)' : 'none',
      }}>
        <span style={{ fontSize: 12, color: '#888' }}>{level}</span>
      </div>
      <code style={{ fontSize: 11 }}>--akds-elevation-{level}</code>
    </div>
  ))}
</div>
